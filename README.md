# fundraisingAnalytics

# Getting Started
## Prerequisites:
Interest  in Analytics and basic exposure to the domain.

Video 1 takes you from basics to an intermediate level.
## Software:
Power BI Desktop (latest version) - Download free
Python 3.9+ (for data generation) - Download
Git (optional, for cloning repository)


This repository suports the two comprehensive Power BI walkthroughs on youtube designed to demonstrate advanced
analytics capabilities for nonprofit fundraising.
Data Assests and faker code for use with walkthrough videos are provided for a quick start

# Why you should engage with ths material?
YouTube educational content (teaching Power BI to aspiring analysts)

Professional development (structured learning for data practitioners)

# Walkthrough_1 covers 
Executive dashboard design for fundraising stakeholders
Donor segmentation using RFM methodology

Campaign performance tracking and KPI visualization

Retention analysis and lifetime value calculations

Stakeholder communication through insights

# Walkthrough_2 covers
Multi-source data integration (CRM, digital, financial systems)

Data quality management and validation frameworks

Predictive modeling for donor lapse risk

Gift propensity scoring for upgrade campaigns

Advanced DAX and time intelligence
